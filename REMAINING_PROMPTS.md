# Remaining Exercise Prompts

This file contains the prompts for generating the missing 'low-poly 3D male mannequin' images for the Peak application.

**Global Style Guidelines:**
- **Subject:** Male mannequin, athletic build.
- **Appearance:** Grey skin tone, low-poly geometric faceted skin.
- **Appirel:** Teal (#00BFA5 approx) geometric tank top and matching shorts.
- **Style:** Low poly, faceted, clean, modern, minimal.
- **Background:** Pure white.
- **Lighting:** Professional studio lighting, soft shadows.
- **View:** Generally front or 3/4 view, best angle to show technique.
- **Resolution:** 1024x1024 or higher.
